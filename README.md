**Welcome to MistPro**

**About**

This project contains everything you need to run your MistPro app locally.

**Edit the code in your local development environment**

Make your changes and push to the repo to update your project.

**Prerequisites:** 

1. Clone the repository using the project's Git URL 
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Create an `.env.local` file and set the right environment variables

```
VITE_APP_ID=your_app_id
VITE_APP_BASE_URL=your_backend_url
```

Run the app: `npm run dev`

**Publish your changes**

Push your changes to the main branch to publish.
